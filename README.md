* Typescript
* Styled-Components
* OpenWeatherMap API for weather data
* Google Geocoding API for coordinates data

	
## Setup :computer:

To run this project, clone then install it locally using yarn:

```
$ git clone https://github.com/lzizn/Weather.git
$ cd Weather
$ pnpm i
$ pnpm run start
```
